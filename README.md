# HR-Attrition-Analysis-PowerBI
HR Attrition Analysis dashboard using Power BI

## Project Overview
Employee attrition is a critical challenge for organizations, as it impacts productivity, hiring costs, and workforce stability.  
This project focuses on *HR Attrition Analysis using Power BI* to understand employee turnover patterns and identify key factors influencing attrition.

The dashboard helps HR teams and management make *data-driven decisions* to improve employee retention and workforce planning.


## Project Objectives
The main objectives of this project are:
- To analyze overall employee attrition trends
- To calculate key HR KPIs such as attrition rate
- To identify departments and job roles with higher attrition
- To study attrition patterns based on gender, age group, and experience
- To build an interactive and user-friendly HR dashboard in Power BI

## Dataset Description
The HR dataset used in this project contains employee-level information such as:
- Employee ID  
- Age  
- Gender  
- Department  
- Job Role  
- Education Field  
- Monthly Income  
- Years at Company  
- Job Satisfaction  
- Attrition Status (Yes / No)  

This dataset enables detailed workforce and attrition analysis.

## Data Cleaning & Preparation
Data preparation was performed using *Power Query*. Key steps include:
- Removing duplicate employee records  
- Handling missing and inconsistent values  
- Correcting data types (numeric, text, categorical)  
- Creating calculated columns for:
  - Age Groups  
  - Experience Bands  
- Validating data accuracy before analysis  

## DAX Measures Created
The following DAX measures were created to support analysis:
- *Total Employees*
- *Attrition Count*
- *Active Employees*
- *Attrition Rate (%)*

These measures update dynamically based on slicer selections.

##  Dashboard Features
The Power BI dashboard includes:
- KPI cards for quick overview of workforce metrics
- Interactive slicers for:
  - Department  
  - Gender  
  - Job Role  
  - Education Field  
- Visuals such as:
  - Bar charts for department and job role analysis  
  - Pie and donut charts for attrition distribution  
  - Tables for detailed employee insights  
- Clean and intuitive layout for easy interpretation  


##  Key Insights
- Certain departments and job roles experience higher attrition
- Younger employees and low-experience groups show higher turnover
- Insights highlight areas requiring HR intervention and retention strategies



##  Business Value
This HR Attrition Analysis dashboard provides value by:
- Helping HR teams identify high-risk attrition segments
- Supporting strategic workforce planning
- Reducing employee turnover through targeted actions
- Enabling management to make informed HR decisions
